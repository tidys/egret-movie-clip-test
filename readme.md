## 注意事项
白鹭引擎在 5.3.6 版本中加入了对 Webpack 的支持，需要安装白鹭编译器，遂放弃该版本，使用了低于5.3.6的版本：[5.2.22](https://github.com/egret-labs/egret-core/archive/refs/tags/v5.2.22.zip)

[文档](https://egret-docs.pages.dev/)

```
"build": "egret build",
"run": "egret run",
"publish": "egret publish"
```
